CryptoGame

Learn. Play. Practice. Cryptography.

Cryptogame offers two things.

A text-based Interactive Storyline which engages the user in a chapter-wise series of cipher challenges, on solving which, the player progresses towards solving a mystery.

A suite of 20+ Cryptography related Tools for a broad selection of ciphers - from classical (Caesar, Vigenere, etc) to modern (RC4, DES and RSA).



Purpose:
1. Engage and entertain the players via a realistic and fascinating storyline, and a set of highly specialized and interconnected cipher challenges.
2. To provide a set of Cryptography tools, using which teachers can demonstrate, and students can practice various cryptography techniques.

Tools Used in building: PHP, MYSQL, JS, HTML, CSS.

INSTALLATION

HOW TO INSTALL
---------------------------
Step 1: Install and run the web server stacks like WAMP 2.4+ (download here), XAMPP, LAMP, etc. (fulfilling the minimum requirements)

Step 2: Put this game folder on the web server (example, in the folder: "C:\wamp\www")

Step 3: Open the folder using web browser (example, url: "localhost/CryptoGame" )

Step 4: Follow the Installation on the web page.

(note:
you may need to give 'enough' permissions to the files and subfolders, when you put the main folder on web server: example, in linux, using commandline, go to that directory and run the command: "chmod 755 * -R". Then, also run the command: "chmod 666 connection/connection_config.php")


MINIMUM REQUIREMENTS
-------------------------------------------
A Web Server (eg. WAMP 2.4+, XAMPP, etc.) having the following or higher versions:

PHP 5.4.12
MySQL 5.6.12
Apache 2.4.4
